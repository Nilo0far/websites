# 🐶 TinDog Landing Page

A modern and responsive landing page for a dog dating app built with **HTML, CSS, and Bootstrap**.

TinDog helps dogs meet new and interesting dogs nearby.

## ✨ Features

* 📱 Fully responsive design
* 🎨 Clean and modern UI
* 🧩 Bootstrap components
* 💳 Pricing section
* ⭐ Testimonials section

## 🛠 Technologies

* 🌐 HTML5
* 🎨 CSS3
* ⚡ Bootstrap
